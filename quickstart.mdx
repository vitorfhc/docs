## Manual installation

If you prefer to build and install Gecko manually, follow these steps.

<Info>
  **Prerequisites**:
  - [Node.js](https://nodejs.org/) (version 14 or higher)
  - [npm](https://www.npmjs.com/) (comes with Node.js)
  - Git
</Info>

<Steps>
<Step title="Clone the repository">

Clone the Gecko repository from GitHub:

```bash
git clone https://github.com/vitorfhc/gecko.git
cd gecko
```

</Step>

<Step title="Install dependencies">

Install the required npm packages:

```bash
npm install
```

</Step>

<Step title="Build the extension">

Build the extension for your browser:

```bash
# For Chrome
npm run build:chrome

# For Firefox
npm run build:ff
```

The build process will create a `dist` folder containing all the necessary files for the extension.

</Step>

<Step title="Load the extension">

### Chrome

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (toggle switch in the top-right corner)
3. Click **Load unpacked**
4. Select the `dist` folder inside the `gecko` directory

### Firefox

1. Open Firefox and navigate to `about:debugging#/runtime/this-firefox`
2. Click **Load Temporary Add-on**
3. Navigate to the `dist` folder and select the `manifest.json` file

</Step>
</Steps>

The extension should now be loaded and visible in your browser toolbar.
